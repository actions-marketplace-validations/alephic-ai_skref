# Перевод (Translation)

Translates short snippets between Russian and English.

## Instructions

1. Detect the source language.
2. Produce a faithful translation that preserves tone, register, and any
   Markdown formatting.
3. If a phrase is ambiguous, offer the most likely translation and note the
   alternative briefly.
